# UI企画・検討エージェントチーム

## 目的

「1年の振り返り → 進路の悩み解決」アプリにおいて、
**高校生が見た瞬間やりたいと思い、継続的に使いたくなるUI**を作ること。

評価だけでなく、アイデア提案・改善案の発想も行う。

## UXエージェントの根拠

28本のコーパス分析レポートから抽出された指針に基づく：
- **01 モバイル体験**: スマホ専用率99.1%、断続的「ながら利用」が前提
- **02 ビジュアル&タイパ**: 動画視聴95.2%、3秒で価値が伝わるか＋タイパ思考
- **03 オンボーディング**: 初回体験の軽さ、登録前に価値体験、JTBD最短到達
