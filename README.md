# 高校生向けキャリア支援アプリ デモ集

高校生のキャリア形成を支援する20個のアプリケーションのインタラクティブデモです。
各HTMLファイルをブラウザで開くと動作を確認できます。

---

## Demo 1: スキルクエスト（`demo1_skill_quest.html`）

日常の活動をRPGクエストに変換し、AIがスキルを自動抽出・レベルアップ表示するアプリ。

- **キャラ選択** — 4体のキャラクターから相棒を選択
- **クエスト入力** — 「部活で後輩に走り方を教えた」のように1行で活動を記録
- **スキル自動抽出** — AIが「リーダーシップ +15XP」のようにスキルを抽出し、レベルアップ演出を表示
- **レーダーチャート** — 6軸（リーダーシップ・創造性・協調性・問題解決力・コミュニケーション・学習意欲）でスキルを可視化

---

## Demo 2: 進化する相棒AI（`demo2_ai_companion.html`）

毎日の短いやりとりからユーザーの個性を学習し、「自分だけの相棒」に進化するAIキャラクター。

- **相棒選択** — 4タイプ（元気な友達・クール先輩・癒し系・ミステリアス知的）から選択し、名前をつける
- **チャット** — 日常のつぶやきに共感・褒め・質問で返すAIとの対話体験
- **気づきレポート** — 会話データから「あなたはチームで何かを達成することにやりがいを感じるタイプ」のような気づきを提示
- **成長ダイアリー** — 日々の会話をAIが自動で成長記録として整理

---

## Demo 3: オンラインメンター相談室（`demo3_mentor_room.html`）

AIメンターに24時間いつでも悩みを相談できるアプリ。共感→思考整理→アクション提示の3段階で対話。

- **メンター選択** — 4スタイル（優しいお姉さん・頼れる先輩・落ち着いた大人・フレンドリーな友達）
- **カテゴリ選択** — 進路・学校生活・将来への不安・自分の強み・ただ聞いてほしい
- **3フェーズ相談** — 共感→整理→アクションの段階的な対話フロー
- **相談サマリー** — 相談内容のまとめと「次にやること」チェックリストを自動生成

---

## Demo 4: 価値観マッピング（`demo4_values_mapping.html`）

「自分が本当に大切にしていること」を10軸で可視化し、企業・大学の文化とマッチングするアプリ。

- **トレードオフ質問** — 「給料が高いけど残業多い」vs「定時で帰れる」のようなスワイプ形式の二択質問
- **10軸レーダーチャート** — やりがい・安定性・人間関係・自律性・社会貢献・成長機会・給与・創造性・チームワーク・WLバランスで可視化
- **企業マッチング** — コサイン類似度ベースでマッチ度を算出し、理由とともに表示
- **友達比較** — 2つのレーダーチャートを重ねて価値観の違いを比較

---

## Demo 5: 企業1日疑似体験（`demo5_work_experience.html`）

企業の1日をAIロールプレイ＋タスク体験でシミュレーション。「この仕事、自分に合うかも？」を体感的に理解。

- **職種選択** — 営業・製造・IT・飲食・建設・介護の6職種から選択
- **タイムライン** — 朝礼→顧客訪問→昼休み→提案書作成→上司報告→振り返りの1日の流れ
- **ロールプレイ** — 上司・顧客などのAI NPCとの対話。3択の選択肢から回答し、フィードバックを受ける
- **適性スコア** — タスク遂行力・対人コミュニケーション・ストレス耐性・正確性・創造性をグラデーション表示

---

## Demo 6: 価値観動画ソート（`demo6_value_video_sort.html`）

価値観が反映された動画をスワイプして好みの順にソートし、自分の価値観を可視化するアプリ。

- **スワイプ判定** — カード型UIで動画を「LIKE」「NOPE」で直感的に判定
- **リアルタイム進捗** — プログレスバーで進行状況を表示
- **価値観ランキング** — 最終結果画面で価値観の優先順位をランキング表示
- **フィードバック演出** — フラッシュオーバーレイと音声で直感的な操作感

---

## Demo 7: ムードボード自己診断（`demo7_mood_board.html`）

気になる雰囲気やシーンをタップして選択し、自分のムードプロフィールを診断するアプリ。

- **マスタリーレイアウト** — 複数の雰囲気カード（風景・場所）を表示
- **インタラクション** — カード選択時にチェック印とリップルアニメーション
- **5つ以上選択** — で結果画面へ移行
- **診断結果** — キーワード、フィーリング、アニメーション付きで表示

---

## Demo 8: あるある行動診断（`demo8_behavior_quiz.html`）

タイマー付きの心理テスト形式でユーザーの行動パターンを診断するアプリ。

- **タイマー付きクイズ** — 進捗ドットと円形タイマーで各問題に時間制限
- **スコアリング** — 複数の行動パターンをスコアリングして結果分析
- **結果ダイアグラム** — 性格診断や行動傾向をビジュアル表示
- **アニメーション演出** — 楽しい診断体験を実現するリッチUI

---

## Demo 9: 音声ジャーナリング（`demo9_voice_journaling.html`）

マイク録音でジャーナリング内容を音声で記録し、感情分析とプロンプトを提供するアプリ。

- **録音ボタン** — グラデーション背景の大型ボタンで音声録音開始/停止
- **ウェーブフォーム** — リアルタイム波形表示で録音中の視覚フィードバック
- **ダッシュボード** — 記録日数・総時間などの統計情報を表示
- **デイリープロンプト** — 毎日異なる問いかけで思考を深堀り

## Demo 10: AI深掘りダイアログ（`demo10_ai_deep_dialogue.html`）

「ミラー」というAIとの対話を通じて気づきや感情を深掘りするアプリ。

- **チャット対話** — AIと自然な会話を交換
- **感情グラフ** — 対話内容の感情変化を分析・可視化
- **インサイトカード** — 重要な気づきを抽出して表示・保存
- **プライバシー保護** — 安全なコミュニケーション環境を強調

## Demo 11: 仕事現場スワイプ（`demo11_work_scene_swipe.html`）

カード式スワイプで様々な職業の現場シーンを評価し、自分の適性を診断するアプリ。

- **スワイプ操作** — カードを「いいね」または「パス」に分類
- **現場カード** — 職業名・業界タグ・現場説明・シーン動画を表示

- **適性診断** — 最終結果画面で診断結果を確認

## Demo 12: 職場環境イメージ選択（`demo12_workplace_image.html`）

- **グリッドレイアウト** — 2〜3列で職場環境カード（オフィス・農場・病院など）を表示

- **レーダーチャート** — 詳細マッチリストを含む結果分析
